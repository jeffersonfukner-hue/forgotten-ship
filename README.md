# A1 Game Academy

> Aprenda Python construindo projetos reais.

A A1 Game Academy é um projeto focado no ensino de Python através do desenvolvimento de jogos completos.

## Objetivos

- Desenvolver jogos completos em Python.
- Ensinar engenharia de software através de projetos reais.
- Produzir documentação durante o desenvolvimento.
- Gerar naturalmente cursos, ebooks e documentação técnica.

## Estrutura

```text
├── metodologia/
├── infraestrutura/
├── projetos/
└── recursos/
```
