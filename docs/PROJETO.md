# Forgotten Ship
### Documento de Projeto

## Missão
Construir um jogo completo em Python, seguindo a metodologia da
A1 Game Academy — usando o desenvolvimento como veículo de aprendizado,
documentação e futura criação de curso.

## Conceito Atual
Jogo de sobrevivência estilo Horde, com restauração de sistemas da nave
como mecânica de defesa. Ver `VISAO.md` para detalhes completos.

## Tecnologias
- Python
- Pygame CE
- Git / GitHub

## Organização
- Desenvolvimento por Sprints (uma branch por Sprint).
- Toda decisão importante é documentada na Sprint correspondente.
- Todo bug relevante é registrado no momento em que é encontrado.
- Metodologia oficial: `METODOLOGIA.md` (repositório `a1-game-academy`).

## Documentos Relacionados
- `VISAO.md` — conceito e experiência do jogo
- `ROADMAP.md` — fases de desenvolvimento do jogo
- `docs/sprints/` — histórico técnico Sprint a Sprint
